# OthelloCloud Frontend

This is the Expo frontend for OthelloCloud.

## Start locally

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npm run start
   ```

3. Open the web build if needed

   ```bash
   npm run web
   ```

## Navigation

The app uses React Navigation for in-app navigation. Main screens live in `src/screens/`.
